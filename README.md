# ZERO-DAY-Dashboard

A dashboard application to manage your college and external courses and track your learning progress.

---

## Project Structure

- `src/` - Source files (CSS, JS)
- `assets/` - Icons and images
- `dist/` - Compiled output files
- `public/` - Public assets
- `Screen Shots/` - Example screenshots
- `package.json` & `package-lock.json` - Node.js dependencies
- **Note:** `node_modules/` is not included in Git. Run `npm install` to generate it locally.

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Yasser-Fekry/ZERO-DAY-Dashboard.git
cd ZERO-DAY-Dashboard
````

2. Install dependencies:

```bash
npm install
```

3. Run the application:

```bash
npm start
```

---

## Usage

* Add and manage college courses
* Add and manage external/self-learning courses
* Track progress on the dashboard
